# 📋 카카오페이지 이용권 장부 (Kakaopage Ticket Ledger)

카카오페이지 **이용권 사용 내역** 페이지를 저장한 HTML 파일을 업로드하면, 실제로 캐시가 나간 금액과 이용권 종류·작품별 사용 내역을 자동으로 집계해주는 웹 도구입니다.

서버 없이 **브라우저 안에서만** 동작하며, 결과는 영수증 형태의 확인서로 만들어 PNG/PDF로 저장할 수 있습니다.

> ⚠️ 카카오페이지와는 무관한 **개인 제작 비공식 도구**입니다.

> 이 페이지는 [리디 결제 장부](https://github.com/starsrebirthyou/my-ridi-order) 레포지토리의 카카오페이지 버전입니다.

---

## ✨ 주요 기능

- **실 결제 금액 자동 계산**: 업로드한 이용권 내역에서 실제로 캐시가 지출된 금액만 합산합니다.
- **무료 이용권 자동 제외**: 아래 이용권은 금액 계산에서 자동으로 제외되고, 건수만 별도로 표시됩니다.
  - 기다무 대여권
  - 선물 무료 대여권
  - 신규혜택 무료 대여권
- **작품 단위 자동 병합**: 같은 작품(웹툰/웹소설)에 대한 여러 결제 항목을 하나로 묶어 합산합니다. (제목이 같으면 동일 작품으로 취급하며, 웹툰/웹소설 분리 기능은 개발 중입니다.)
- **이용권 종류별 비중 집계**: 소장권/대여권/기타 등 이용권 종류별 결제 비율과 금액을 표와 그래프 바로 보여줍니다.
- **다중 파일 업로드 지원**: 여러 시점에 나눠 저장한 파일이 있다면 한 번에 여러 개를 올려서 합산할 수 있습니다.
- **결과 내보내기**: 완성된 확인서를 PNG 또는 PDF로 저장할 수 있습니다.

---

## 🖥️ 사용 방법

### 1. 이용권 내역 저장하기

1. 브라우저에서 카카오페이지 이용권 내역 페이지(`page.kakao.com/history/ticket/`)에 접속합니다.
2. 내역이 길다면 스크롤을 끝까지 내려 전체 항목을 로딩합니다.
3. `Ctrl + S` (다른 이름으로 저장)를 눌러 페이지를 HTML 파일로 저장합니다.

### 2. 파일 업로드

1. `index.html`을 브라우저에서 엽니다.
2. 저장해둔 `.html` 파일을 업로드 영역에 끌어다 놓거나 클릭해서 선택합니다.
3. 여러 시점에 나눠 저장한 파일이 있다면 한 번에 여러 개를 선택해도 됩니다.

### 3. 결과 확인 및 저장

- 업로드가 끝나면 총 결제 금액, 결제 건수, 제외된 무료 이용권 건수, 작품 수, 이용권 종류별 비중, 작품별 결제 내역이 자동으로 나타납니다.
- **확인서 PNG로 저장**: 상단 요약 확인서만 이미지로 저장합니다.
- **전체 내역 PDF로 저장**: 전체 결과를 PDF로 저장합니다.
- **전체 초기화**: 업로드한 데이터를 모두 지우고 처음 상태로 되돌립니다.

---

## 🔒 데이터 처리 방식

업로드한 파일은 **서버로 전송되지 않고 사용자의 브라우저 안에서만** 파싱·계산됩니다. 페이지를 새로고침하면 모든 데이터가 초기화되며, 원본 파일 자체도 저장되지 않으므로 다시 확인하려면 파일을 다시 업로드해야 합니다.

## 🛠️ 기술 스택

- 순수 HTML / CSS / JavaScript (별도 빌드 과정 없음)
- [html2canvas](https://github.com/niklasvh/html2canvas) — 확인서를 이미지로 캡처
- [jsPDF](https://github.com/parallax/jsPDF) — 확인서를 PDF로 저장

## 📁 파일 구성

| 파일 | 설명 |
|---|---|
| `index.html` | 페이지 마크업과 업로드/파싱/집계/내보내기 로직 |
| `style.css` | 영수증 스타일 UI 디자인 |

## 🚀 로컬에서 실행하기

별도 설치 없이 `index.html`을 브라우저로 바로 열면 됩니다.

---

## ⚠️ 주의사항 및 한계

- 카카오페이지 페이지 구조가 변경되면 파싱 로직이 정상 동작하지 않을 수 있습니다.
- 제목이 같은 웹툰과 웹소설은 현재 동일 작품으로 합산됩니다. (분리 기능 개발 중)
- 이 도구가 계산한 금액은 참고용이며, 실제 결제 내역은 카카오페이지 공식 페이지에서 확인하시기 바랍니다.
