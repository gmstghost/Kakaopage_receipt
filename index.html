기다무 대여권 · 선물 무료 대여권 · 신규혜택 무료 대여권은 금액 계산에서 자동으로 제외됩니다.
제목이 같은 웹툰과 웹소설은 같은 작품으로 취급됩니다. (분리 기능 개발 중)

사용법 / 도움말 펼치기

1. 데이터 넣는 방법

이 페이지는 HTML 파일 업로드만 지원합니다. 카카오페이지 page.kakao.com/history/ticket/ 페이지를 브라우저에서 Ctrl+S(다른 이름으로 저장)한 뒤 올려주세요. 내역이 길면 스크롤을 끝까지 내려 전체를 로딩한 다음 저장하는 걸 추천해요. 여러 시점에 나눠 저장했다면 한 번에 여러 개 올려도 됩니다.

2. 자동으로 처리되는 것들

  • 기다무 대여권 / 선물 무료 대여권 / 신규혜택 무료 대여권 → 총 결제액 계산에서 자동 제외 (건수는 따로 표시)
  • 같은 작품의 여러 결제 항목 → 자동으로 묶어서 금액 합산
  • 이용권 종류(소장권/대여권/기타)별 비중 자동 집계

3. 데이터는 어디에 저장되나요

업로드한 파일은 서버로 전송되지 않고 사용자의 브라우저 안에서만 처리됩니다. 새로고침하면 초기화되고, 원본 파일 자체는 저장되지 않으니 다시 열면 파일을 다시 올려야 해요.
